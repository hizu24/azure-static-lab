経路ルール

  • Microsoft 365系：直接通信
  • 社内 + 社内システム：社内LAN経由
  • 社外 + 社内システム：GlobalProtect → Prisma Access → 社内システム
  • 特定サイト：プロキシ経由
  • それ以外：Prisma Access経由

案件に置き換えると

  • 社内：会社ネットワークにいる端末
  • 社外：自宅・外出先・スマホ回線などから使う端末
  • GlobalProtect：端末からPrismaへつなぐエージェント
  • Prisma Access：通信を検査・制御するクラウド側の基盤
  • 直接通信：Prismaを通さず、外部サービスへ出す通信
  • 社内LAN経由：社内にいる時だけ使える内部経路
